    clock_replacer

    introduction

    ClockReplacer 类内部维护了一个 frame 的集合，集合大小和缓冲池的大小一致。

    需要实现四个函数:

    • Victim(T*) : Starting from the current position of clock hand, find the first frame that is both in the ClockReplacer and with its ref_flag_flag flag set to false. If a frame is in the ClockReplacer, but its ref_flag_flag flag is set to true, change it to false instead. This should be the only method that updates the clock hand.
    • Pin(T) : This method should be called after a page is pinned to a frame in the BufferPoolManager. It should remove the frame containing the pinned page from the ClockReplacer.
    • Unpin(T) : This method should be called when the pin_count of a page becomes 0. This method should add the frame containing the unpinned page to the ClockReplacer.
    • Size() : This method returns the number of frames that are currently in the ClockReplacer.

    添加了三个变量:

    victim

    这个函数是将ClockReplacer中满足条件的frame移出内存,条件如下

    • 在Clockreplacer中,即当前没有任何的线程访问该frame
    • ref_flag_flag_flag == false 时间替换策略的本质就是,ref_flag_flag_flag表示当前帧是否被修改过,victim()在遍历所有帧时,如果该帧的ref_flag_flag_flag == true,那么表示最近被访问过,如果将其改为false,并且下次clock_hand访问到该frame时仍然为false表示其最近并未被访问过,即可移出内存

    pin

    这个函数是将指定的frame调入BufferPool,

    unpin

    这个函数是线程调用该frame完毕,将其调入Clockreplacer

    Size

    这个函数获取当前ClockReplacer中所有frame的数量
