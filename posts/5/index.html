html记忆重构(四)

发表于 2022-05-30 更新于 2022-07-06 分类于 Html5 and Css
那一刻我们的吻,像轰鸣的月相

后代选择器

可以精确地选择到标签中的标签.例如有多个li,我只想修改无序表中的li的样式,只需要在类选择器li的前面加上ul,例如输入
ul li {
color: pink;
font-size: xx-large;
}
就可以将无序表中的字体颜色和字体大小修改成想要的样式

如果只想修改li中的某一类标签,可以继续选择,只需要在后面加上标签名即可

如果有多个ul,而我们只想修改某一个ul中的li,只需要给该ul添加一个类,同时将ul改成.classname即可

.test li a {
        text-decoration: none;
        color: orange;
    }

子选择器

格式如下:

.test>div {
    font-size:25px
    line-height:25px
}

在父类的后面加上>,则选择最近一级的子元素,亲儿子元素

<div class="test">
<div>亲儿子</div>
<div><div>孙子</div><div>
</div>

并集选择器

要给多个类选择器添加同样的属性,只需要用,即可,如下

    div,
    p,
    .test li {
        color: red;
    }

链接伪类选择器

以链接为例,可以修改其各个状态的属性

    a {
         text-decoration: none;//去掉下划线
    }
     /* 未访问过的链接  a:link*/
    a:link {
        color: black;
    }

    /* 访问过的链接  a:visited*/
    a:visited {
        color: #576690;
    }

    /* 鼠标经过的链接 a:hover */
    a:hover {
        color: green;
    }

    /* 鼠标按下还未弹起的链接 a:active */
    a:active {
        color: aqua;
    }

这里只修改了链接的各种状态下的颜色.

focus语法

为input标签添加focus属性,在聚焦该input标签时的属性]

 input:focus {
        color: pink;
        background-color: #576690;
    }

未聚焦时,文本框为白色背景,聚焦后,背景颜色变为#576690,字体颜色变为pink

块级元素/行内元素/行内块元素

块元素典型的是div,可以转换为其他两种形式.

display: block;转换为块元素
display: inline;转换为行内元素
display: inline-block;转换为行内块元素

tips: 将行高设置为块的高度即可实现单行文字垂直居中

背景相关

背景颜色

background-color: #576690

默认为transparent

背景图片

background-image: url(绝对路径/相对路径/网址)
