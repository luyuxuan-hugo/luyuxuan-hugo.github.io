    html记忆重构5

    发表于 2022-07-11 更新于 2023-06-23 Waline： 本文字数： 837 阅读时长 ≈ 1 分钟

    三大特性

    层叠性

    设置相同的样式,一个样式就会覆盖另一个冲突的样式,遵守就近原则,不冲突的不覆盖

    继承性

    子标签会继承父标签的的某些样式,如 text-, font-, line-, 以及color属性

    优先级

    当同一个元素指定多个选择器,根据选择器的权重来执行

    继承或 * : 0,0,0,0 ;
    元素选择器 : 0,0,0,1 ;
    类选择器/伪类选择器 : 0,0,1,0 ;
    ID选择器 : 0,1,0,0 ;
    行内样式style=”” : 1,0,0,0 ;
    !important : 无限大 ;

    复合选择器有权重叠加的问题

    盒子模型

    边框

    border-width

    边框的粗细

    border-style

    边框样式,默认为none;

    border-color

    边框颜色

    复合写法

    没有先后顺序

    可以只设定边框的一条边

    border-collapse

    边框合并

    边框会影响原本盒子的大小

    内边距padding

    内边距,即内容与边框之间的距离

    可以单独写每个方向的内边距

    也可以复合写,可以有四种写法

    padding也会影响盒子的大小
    如果本身没有指定width和height值,padding不会撑开盒子
